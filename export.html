‚ùì FAQ

Can I use this as a simple HTML website?

No, this is a React application that requires a build process. However, after building (npm run build), you get static HTML/CSS/JS files in the /dist folder that can be hosted anywhere.

Do I need a backend/database?

Currently, the app uses mock data for demonstration. For production use, you'll need to implement a backend with Supabase (recommended) or another database solution.

How much does it cost to run?

  • Hosting: Free (Netlify/Vercel free tier) or $5-50/month
  • Supabase: Free up to 500MB, then $25/month
  • Stripe: 2.9% + $0.30 per transaction
  • Domain: $10-15/year

Can I customize the design?

Yes! All styling is done with Tailwind CSS. You can easily change colors, fonts, spacing, and layouts by editing the Tailwind classes or the theme.css file.

Is this mobile-friendly?

Yes, the entire application is built with mobile-first responsive design. It works perfectly on phones, tablets, and desktops.
